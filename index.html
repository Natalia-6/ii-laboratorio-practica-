Preguntas Introductivas

  • ¿Que es un framework?

    Es el esquema o estructura que se establece y que se aprovecha para desarrollar y organizar un software determinado. Esta definición, algo compleja, podría resumirse como el entorno pensado para hacer más sencilla la programación de cualquier aplicación o herramienta actual.

  • ¿Que es un bootstrap 5 y cual es su origen?

    Es el framework más popular para el desarrollo de interfaces web con un diseño moderno, adaptadas a las pantallas de los dispositivos.

    Origen. Bootstrap, originalmente llamado Blueprint de Twitter, fue desarrollado por Mark Otto y Jacob Thornton de Twitter, como un marco de trabajo (framework) para fomentar la consistencia entre las herramientas internas. ... Como resultado, decenas de temas se han introducido en el marco de trabajo.

  • ¿Que es diseño responsive en páginas web?

    El diseño web responsive o adaptativo es una técnica de diseño web que busca la correcta visualización de una misma página en distintos dispositivos. Desde ordenadores de escritorio a tablets y móviles.

  • ¿Que son las github pages?

    Es la función de github que permite generar un link para poder hacer publica nuestra pagina web

  • ¿Cuál es la diferencias entre elemento y compoenente web?

    Los componentes web son un conjunto de características que actualmente están siendo añadidas por el W3C a las especificaciones HTML y DOM de forma que permite la creación de widgets o componentes reutilizables en documentos y aplicaciones web.​

  • ¿Cuál es la importancia de documentar o comentar código?

    Añade suficiente información como para explicar lo que hace, punto por punto, de forma que no sólo los ordenadores sepan qué hacer, sino que además los humanos entiendan qué están haciendo y por qué.

  • ¿Por qué existe una sección de atribución en los sitios web?

    Para dar créditos a los creadores y reconocimiento y para evitar algun tipo de copyright

Una pequeña reflexión sobre mi proyecto y mi carrera










REFLEXIÓN PERSONAL


Personalmente este proyecto de las prácticas me ha servido mucho ya que he explorado bastantes componentes y pude usar mi creatividad.

Este proyecto de laboratorio fue una buena manera de poner en práctica todo lo que aprendimos durante los 3 bloques.

He aprendido a usar los lenguajes de HTML y CSS, que en la actualidad son muy solicitados,

el hecho de que en un año a pesar de las dificultades y que todo haya sido virtual, me sirvió ha ser alguien mas independiente.

Obviamente hubieron situaciones en las cuales si me costaba entender pero fue algo que tambien me ayudo a mejorar, aprender de los errores.

Como estudiante siento que en poco tiempo podré crear una página web por mi misma, hasta el momento no lo he logrado pero creo que todo lo que he apredido me servirá para la universidad y hasta incluso para el trabajo.

Y por último aunque fué un proyecto largo y un poco desafiante, estoy bastante satisfecha por el resultado final :)
